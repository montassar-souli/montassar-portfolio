# Project Summary
This project is a modern portfolio website designed for developers to effectively showcase their skills and projects. It emphasizes an engaging user experience through visually appealing UI/UX design and smooth animations. The portfolio features a comprehensive "About Me" section, a contact page integrated with EmailJS for message delivery, and a projects section that highlights both public GitHub projects and private client work with a focus on business impact.

# Project Module Description
The main modules of this project include:
- **Hero Component**: The introductory section featuring an animated background and call-to-action buttons.
- **About Component**: A detailed section about Montassar Souli, showcasing his skills, professional experience, education, and personal interests.
- **AboutRecap Component**: A concise overview placed after the hero section on the homepage, highlighting key information such as expertise and achievements, along with animated statistics.
- **Contact Component**: A professional contact page with a form using Zod for validation and EmailJS for sending messages, featuring various states for user feedback.
- **Projects Component**: A section showcasing various projects with detailed descriptions, technologies used, and their business impact.
- **ProjectsRecap Component**: A summary section on the homepage highlighting top projects and key statistics.
- **Footer Component**: A newly redesigned footer section featuring modern design elements, enhanced user experience, and interactive features.

# Technology Stack
- **React**: JavaScript library for building user interfaces.
- **Next.js**: React framework for server-side rendering and static site generation.
- **Tailwind CSS**: Utility-first CSS framework for styling.
- **Framer Motion**: Animation library for React that enables smooth animations and transitions.
- **Zod**: TypeScript-first schema declaration and validation library.
- **React Hook Form**: Library for managing form state and validation.

# Usage
To set up the project, follow these steps:
1. Install dependencies:
   ```bash
   npm install framer-motion @emailjs/browser zod react-hook-form @hookform/resolvers
   ```
2. Build and run the project:
   ```bash
   npm run build
   npm start
   ```
